# Mr. Robot CTF Walkthrough

This repository contains a detailed walkthrough and technical documentation of the Mr. Robot Capture The Flag (CTF) challenge completed on TryHackMe.

## Project Overview
The goal of this project was to apply practical cybersecurity concepts in a hands-on environment and document the complete attack methodology in a structured and readable format.

## Skills Demonstrated
- Reconnaissance and enumeration
- Linux privilege escalation
- Credential discovery
- Security-focused documentation

## Tools Used
- Nmap
- Linux utilities
- Enumeration tools
